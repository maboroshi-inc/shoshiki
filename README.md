# shoshiki

テキスト整形ユーティリティ

## インストール

```sh
npm install @maboroshi/shoshiki
```

or

```sh
yarn add @maboroshi/shoshiki
```

## 使い方

```js
import { numToCommaSeparated } from '@maboroshi/shoshiki'

console.log(numToCommaSeparated(1234)) // => 1,234
```
